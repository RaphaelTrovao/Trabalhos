Oque é RPG?

O papel dos Dados

Rpg é um estilo de jogo de tabuleiro onde os jogadores interpretam seus personagens, tendo total controle sobre suas ações, que são regidas pelo sistema respectivo, que por sua vez, utilizam dados para definir os aspectos de sorte que envolvem o jogo.

Os dados são parte importante do Rpg, são eles que transformam o jogo em um ambiente justo para todos os envolvidos, aplicando as regras perante todos os jogadores e mestre, neste site, é disponibilizado os dados mais utilizados para rpg, juntamente de uma tabela que armazena suas lançadas recentes.
